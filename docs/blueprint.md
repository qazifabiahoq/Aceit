# **App Name**: AceIt

## Core Features:

- Real-time Speech Analysis: Analyze the user's speech in real-time, providing feedback on structure, clarity, and relevance using an AI Speech agent.
- Vision Analysis: Analyze the user's facial expressions, eye contact, and body language via webcam using an AI Vision agent.
- AI Coaching Orchestration: Coordinate insights from the Speech, Vision, and Voice AI agents using an AI Coach tool to provide holistic feedback.
- Voice Feedback Delivery: Deliver coaching feedback naturally through speech, without interrupting the user's flow using an AI Voice agent.
- Live Transcription: Provide a live transcript of the user's speech during the interview session.
- Session Timer and Question Management: Display a session timer and manage the flow of interview questions.
- Results Summary: Generate a summary of the session results, including overall score, individual agent scores, strengths, and areas for improvement.

## Style Guidelines:

- Background color: Deep dark background (#121212) for a premium, modern feel.
- Primary color: Off-white (#F5F5F5) for text to ensure high contrast and readability.
- Accent color: Electric blue (#7DF9FF) for CTAs, highlights, and agent type color-coding, to convey an AI aesthetic.
- Body and headline font: 'Inter', a grotesque-style sans-serif font, is recommended for a modern, machined, objective look suitable for headlines and body text.
- Use clean, minimalist icons throughout the interface to represent features and agent types; avoid emojis.
- Implement a responsive design with a three-column layout on desktop and a stacked layout on mobile for optimal viewing on all devices.
- Incorporate smooth transitions and subtle hover states to enhance user experience without being distracting.
